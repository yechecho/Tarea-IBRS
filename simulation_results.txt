EXPECTED SIMULATION BEHAVIOR
==========================================================================

─────────────────────────────────────────────────────────
PHASE 1 — Initialization (t = 0 to 0.05 s)
─────────────────────────────────────────────────────────
Description:
  System starts from rest. The PLL begins tracking the grid angle.
  Converter output voltages are near zero until the PLL locks.

Expected Observations:

  Grid Voltages (Va, Vb, Vc):
    • Sinusoidal, 60 Hz, amplitude ≈ 179.6 V (peak)
    • Balanced three-phase, 120° phase shift
    • Remain stable throughout simulation

  PLL Output (Theta):
    • Starts at 0 rad, ramps toward grid angle
    • Transient oscillation as the PLL acquires lock
    • PLL natural frequency: ω_n ≈ 70.7 rad/s (11.25 Hz)
    • PLL damping ratio: ζ ≈ 0.707 (critically damped)

  PLL Output (Omega):
    • Initial value: 376.99 rad/s (2π × 60)
    • Transient variation ±5–10 rad/s during acquisition
    • Settles to 376.99 rad/s within ≈ 50 ms

  Filter Currents (Ia, Ib, Ic):
    • Initially zero (or very small)
    • Start flowing as converter begins modulating

─────────────────────────────────────────────────────────
PHASE 2 — PLL Lock-on (t ≈ 0.05 to 0.10 s)
─────────────────────────────────────────────────────────
Description:
  PLL achieves synchronization with the grid. Theta tracks the
  grid voltage angle. Current control begins regulating Id and Iq.

Expected Observations:

  PLL Angle (Theta):
    • Smoothly follows grid angle (linear ramp at 60 Hz)
    • Bounded in [0, 2π] due to modulo operation
    • Phase error < 1° after lock

  DQ Voltages (Vd, Vq):
    • Vd → ≈ 179.6 V (peak phase voltage)
    • Vq → ≈ 0 V (condition for PLL lock)
    • Small ripple at 6× grid frequency (360 Hz)

  DQ Currents (Id, Iq):
    • Follow references set by Droop Control
    • With P_ref = 0, Q_ref = 0: Id_ref ≈ 0, Iq_ref ≈ 0
    • Settling time: < 50 ms (inner loop bandwidth ≈ 667 rad/s)

─────────────────────────────────────────────────────────
PHASE 3 — Steady-State Operation (t = 0.10 to 0.50 s)
─────────────────────────────────────────────────────────
Description:
  System operates in steady state. With default references
  P_ref = 0 W, Q_ref = 0 VAR (zero-power injection).

Expected Observations:

  Grid Currents (Ia, Ib, Ic):
    • Small residual currents (reactive charging of filter L)
    • Amplitude < 1 A for zero power reference
    • Balanced and sinusoidal at 60 Hz

  DQ Frame Currents:
    • Id ≈ 0 A  (no active power)
    • Iq ≈ 0 A  (no reactive power)
    • Ripple amplitude < 0.5 A (due to PWM harmonics)

  Active Power (P):
    • ≈ 0 W in steady state (P_ref = 0)
    • Ripple at 6× fundamental = 360 Hz, amplitude < 50 W
    • Low-pass filtered at 5 Hz bandwidth

  Reactive Power (Q):
    • ≈ 0 VAR in steady state (Q_ref = 0)
    • Similar ripple characteristics to P

  Converter Output Voltages (Va, Vb, Vc):
    • PWM-modulated, switching at 5 kHz
    • Fundamental component ≈ grid voltage after filter
    • THD of voltage after filter < 5%

  Gate Signals (6 switches):
    • Complementary pairs (upper/lower per phase)
    • Switching frequency: 5000 Hz
    • Dead-time not modeled (educational simplification)

TUNING GUIDELINES
─────────────────────────────────────────────────────────

Current Controller (Inner Loop):
  • Bandwidth target: ω_cc = R/L = 6.67 rad/s natural → set higher
  • Kp_id = ω_cc * L_filter → for ω_cc = 667 rad/s: Kp ≈ 10 ✓
  • Ki_id = ω_cc * R_filter → Ki ≈ 67 (use 100 for margin) ✓

PLL:
  • Desired lock time < 100 ms → ω_n > 50 rad/s
  • ω_n = sqrt(Ki_pll) → Ki = 5000 → ω_n ≈ 70.7 rad/s ✓
  • Critically damped: Kp = 2*ζ*ω_n → Kp = 2*0.707*70.7 ≈ 100 ✓

Droop Control (Outer Loop):
  • Rule: outer loop bandwidth < inner loop bandwidth / 10
  • Target: outer loop ≈ 10–50 rad/s
  • Kp_droop = 0.01: P error of 1 kW → ΔId_ref = 10 A ✓
