# VSC Grid-Following Control System — Tarea IBRS

Complete MATLAB/Simulink implementation of a **grid-following Voltage Source
Converter (VSC)** control system, including a Synchronous Reference Frame PLL,
dual-axis current control (dq frame), droop-based outer loop, and SVPWM
modulation.

---

## System Specifications

| Parameter | Value |
|---|---|
| Grid Frequency | 60 Hz |
| Grid Voltage (line-to-line RMS) | 220 VAC |
| Converter Power Rating | 10 kW |
| Filter Inductance | 15 mH |
| Filter Resistance | 0.1 Ω |
| DC-link Voltage | 400 VDC |
| DC-link Capacitance | 2200 µF |
| PWM Frequency | 5 kHz |
| Control Sampling Time | 10 µs |

## Subsystem Details

### 1. GridSource
Three sinusoidal voltage sources (120° phase shift, 60 Hz, 179.6 V peak).

### 2. Filter
Per-phase RL filter implemented as transfer functions `1/(L·s + R)`.  
The VSC output voltage is subtracted from the grid voltage to produce
the driving voltage for each phase current.

### 3. abc2dq / abc2dq_curr  (Park Transformation)
Implements the full Clarke + Park transform:

```
α = (2/3)(xa − xb/2 − xc/2)
β = (2/3)(√3/2·xb − √3/2·xc)

xd =  α·cos(θ) + β·sin(θ)
xq = −α·sin(θ) + β·cos(θ)
```

### 4. PLL — Synchronous Reference Frame PLL
Forces `Vq → 0` to achieve phase lock.

```
error = −Vq
Δω   = PI(error)  [Kp=100, Ki=5000]
ω    = ω_grid + Δω
θ    = ∫ ω dt  (mod 2π)
```

**Analytical performance:**
- Natural frequency: ω_n = √5000 ≈ 70.7 rad/s
- Damping ratio: ζ = 100/(2·70.7) ≈ 0.707 (critically damped)
- Lock time: < 100 ms

### 5. DroopControl  (Outer Loop)
Calculates instantaneous three-phase power in the dq frame:

```
P = 1.5·(Vd·Id + Vq·Iq)
Q = 1.5·(Vd·Iq − Vq·Id)
```

Generates current references via droop gains:

```
Id_ref = Kp_droop · (P_ref − P_filtered)
Iq_ref = Kq_droop · (Q_ref − Q_filtered)
```

A 5 Hz low-pass filter smooths the power signals.  
Output currents are saturated at ±1.5× rated peak.

### 6. CurrentControl  (Inner Loop)
Dual PI controllers with cross-coupling decoupling:

```
Vd_ref = PI_Id(Id_ref − Id) − ω·L·Iq + Vd_meas
Vq_ref = PI_Iq(Iq_ref − Iq) + ω·L·Id + Vq_meas
```

Output voltages are saturated at ±Vdc/2 = ±200 V.

**Bandwidth:** ω_cc = Kp/L = 10/0.015 ≈ 667 rad/s (106 Hz)

### 7. dq2abc  (Inverse Park Transformation)
```
α = Vd·cos(θ) − Vq·sin(θ)
β = Vd·sin(θ) + Vq·cos(θ)

Va =  α
Vb = −α/2 + (√3/2)·β
Vc = −α/2 − (√3/2)·β
```

### 8. SVPWM (PWM Generation)
Sinusoidal carrier-based PWM at 5 kHz.  
Normalizes voltage references by `Vdc/2`, compares against triangular
carrier to produce 6 complementary gate signals.

### 9. VSC_Model
Ideal two-level converter: output voltage per phase = `Sa_upper × Vdc − Vdc/2`.

### 10. Visualization
Six oscilloscope blocks:
1. Three-phase grid voltages (Va, Vb, Vc)
2. Three-phase filter currents (Ia, Ib, Ic)
3. DQ voltages (Vd, Vq)
4. DQ currents — measured vs. reference (Id, Iq, Id_ref, Iq_ref)
5. PLL outputs (Theta, Omega)
6. Active and reactive power (P, Q)

## Parameter Tuning

Edit `parameters.m` and re-run `create_VSC_model.m` to rebuild the model
with updated values.

### Change power reference (active power injection):
```matlab
P_ref = 10000;  % 10 kW injection
Q_ref = 0;      % unity power factor
```

### Tighten current loop (faster response):
```matlab
Kp_id = 20;   % higher proportional gain
Ki_id = 200;  % higher integral gain
```

### Adjust PLL bandwidth:
```matlab
Kp_pll = 150;    % faster acquisition
Ki_pll = 10000;  % higher natural frequency
```

## Known Limitations (Educational Model)

1. Dead-time between complementary switches not modeled
2. DC-link treated as ideal voltage source (no capacitor dynamics)
3. No anti-windup on PI integrators
4. No grid impedance model
5. Switch losses not included
